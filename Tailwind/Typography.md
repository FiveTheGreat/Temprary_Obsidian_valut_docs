How to manipulate and customize the text properties

##### Font-Size:

| property    | content |
| ----------- | ------- |
| text-xs     | 12px    |
| text-sm     | 14px    |
| text-base - | 16px    |
| text-lg     | 18px    |
| text-lx     | 20px    |


Hint:
* `sm`  small
* `xs`  extra small
* `base`  normal
* `lg`  long
* `xl`  extra long
* diffreces- 2px 
* line-hight - 4px

Here The default `line-hight:1`, we can adjust line hight `/` using this example

```css
text-xs/5 - 12px,5 line-hight
```
